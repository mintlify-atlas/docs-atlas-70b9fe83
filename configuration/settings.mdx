Django Unfold provides extensive configuration options through the `UNFOLD` setting in your Django settings file. All settings are optional and have sensible defaults.

## Configuration structure

Add the `UNFOLD` dictionary to your Django settings:

```python settings.py
UNFOLD = {
    "SITE_TITLE": "My Admin",
    "SITE_HEADER": "Administration",
    # ... other settings
}
```

## Site branding

<ParamField path="SITE_TITLE" type="string" default="None">
  The site title displayed in the browser tab and admin header.
</ParamField>

<ParamField path="SITE_HEADER" type="string" default="None">
  The main header text displayed in the admin interface.
</ParamField>

<ParamField path="SITE_SUBHEADER" type="string" default="None">
  Subheader text displayed below the main header.
</ParamField>

<ParamField path="SITE_DROPDOWN" type="string" default="None">
  Content for the site dropdown menu in the header.
</ParamField>

<ParamField path="SITE_URL" type="string" default="/">
  URL the site logo/title links to.
</ParamField>

<ParamField path="SITE_ICON" type="string" default="None">
  Path to the icon displayed in the admin interface.
</ParamField>

<ParamField path="SITE_SYMBOL" type="string" default="None">
  Symbol or short text displayed alongside the site title.
</ParamField>

<ParamField path="SITE_LOGO" type="string" default="None">
  Path to the logo image displayed in the admin interface.
</ParamField>

## Display options

<ParamField path="SHOW_HISTORY" type="boolean" default="True">
  Show the history link in the admin interface for objects.
</ParamField>

<ParamField path="SHOW_VIEW_ON_SITE" type="boolean" default="True">
  Display the "View on site" button when available.
</ParamField>

<ParamField path="SHOW_LANGUAGES" type="boolean" default="False">
  Enable language switcher in the admin interface.
</ParamField>

<ParamField path="LANGUAGE_FLAGS" type="object" default="{}">
  Dictionary mapping language codes to flag icons or images.
</ParamField>

<ParamField path="SHOW_BACK_BUTTON" type="boolean" default="False">
  Show a back button in the admin interface navigation.
</ParamField>

## Environment

<ParamField path="ENVIRONMENT" type="string" default="None">
  Environment name displayed in the admin interface (e.g., "Development", "Staging", "Production").
</ParamField>

<ParamField path="ENVIRONMENT_TITLE_PREFIX" type="string" default="None">
  Prefix added to browser tab titles based on environment.
</ParamField>

## Dashboard

<ParamField path="DASHBOARD_CALLBACK" type="string" default="None">

</ParamField>

  <Expandable title="properties">
    <ParamField path="search_models" type="boolean" default="False">

    <ParamField path="search_callback" type="string" default="None">

## Example configuration

```python settings.py
UNFOLD = {
    "SITE_TITLE": "My Admin Panel",
    "SITE_HEADER": "Administration",
    "SITE_URL": "/",
    "SITE_ICON": "/static/icon.svg",
    "SHOW_HISTORY": True,
    "SHOW_VIEW_ON_SITE": True,
    "ENVIRONMENT": "production",
    "SIDEBAR": {
        "show_search": True,
        "show_all_applications": True,
    },
    "TABS": [
        {
            "models": [
                "app.Model1",
                "app.Model2",
            ],
            "items": [
                {
                    "title": "Dashboard",
                    "link": "/admin/",
                },
            ],
        },
    ],
}
```

<Info>
Settings are merged with defaults using deep merging. You only need to specify the settings you want to override.
</Info>
