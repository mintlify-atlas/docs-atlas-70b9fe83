Django Unfold is actively developed and continuously evolving. This changelog tracks all notable changes, new features, bug fixes, and improvements.

## Latest Release

For the complete, up-to-date changelog, visit the [official Django Unfold changelog on GitHub](https://github.com/unfoldadmin/django-unfold/blob/main/CHANGELOG.md).

## Recent Highlights

### v0.82.0 (2026-03-03)

**Features**
- Added UnfoldAdminRelatedFieldWrapperWidget for enhanced form field rendering
- Dropped Python 3.10 support - minimum version is now Python 3.11

**Bug Fixes**
- Centered expand icon for collapsed fieldsets

### v0.81.0 (2026-02-25)

**Features**
- Upgraded to Tailwind CSS 4.2
- Added fullwidth constance page support

**Bug Fixes**
- Fixed array widget attributes handling
- Improved filter spacing
- Fixed JSON editor dark mode scrollbar
- Redesigned checkbox and radio inputs
- Fixed modal overlay for calendar widget

### v0.80.0 (2026-02-16)

**Features**
- Redesigned user avatar component
- Added simple table row hover effects
- Enhanced button component with hover effects

**Bug Fixes**
- Added shadow to default button variant
- Fixed dataset top spacing in tabs with errors
- Improved table hover colors in light mode
- Used app verbose name in header titles

## Version Support

<Note>
Django Unfold follows semantic versioning. We recommend staying up-to-date with the latest release for the best experience and latest features.
</Note>

### Python Support

- **v0.82.0+**: Requires Python 3.11 or higher
- **v0.81.0 and earlier**: Supports Python 3.10+

### Django Support

Django Unfold maintains compatibility with:
- Django 4.2 LTS
- Django 5.0
- Django 5.1 (latest)
